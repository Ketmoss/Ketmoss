## Approche

- Produits **Rails API + React/Next.js** ou **Rails Hotwire** selon le contexte
- Applications **Flutter** pour le mobile cross-platform
- Intégration d'**IA générative** (LLM, RAG, embeddings) dans des produits métier
- Architecture orientée **performances** : Redis, Sidekiq, pgvector, caching
- Déploiement continu avec **Kamal** (self-hosted) ou **Vercel**
